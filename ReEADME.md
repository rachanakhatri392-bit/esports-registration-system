# Esports Registration System

A web-based application developed using HTML, CSS, PHP, and MySQL to manage participant registration for esports tournaments.

## Features
- User registration
- User login
- Admin login
- Edit participant information
- Delete participant records
- Search and manage registrations

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## How to Run Locally
1. Place the project inside `C:\xampp\htdocs\esports`
2. Start Apache and MySQL in XAMPP
3. Import `esports.sql` in phpMyAdmin
4. Open `http://localhost/esports`
